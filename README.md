                   HOW TO UPLOAD A SHARED FILE INTO YOUR GITHUB .
                   -1)CLICK ON CODE AND COPY THE FILE HTTPS
                   -2)OPEN FILE MANAGER AND CLICK ON C OR D DRIVE
                   -3)OPEN COMMAND PROMPT (CMD)
                   -4)TYPE - git clone " paste already copied file ". and ENTER
                   - The cmd will give its name as - Cloning into ' FILE NAME' .whatever it is...
                   -5) TYPE - cd FileName - and Enter
                   -6) Type - git pull - it will show " Already up to date ".
                   -7) Type - "code ." - Code space and Dot.

                   --------------------------------------------------------------------

                   HOW TO UPDATE THE ALREADY PUSHED GITHUB FILE THROUGH IN THE VS CODE.

                   -1) Type - git add .
                   -2) Type - git commit -m "Comments "
                   -3) Type - git push

                   HOW TO RETRIVE RECENTLY UPDATED GITHUB FILE IN VS CODE.
